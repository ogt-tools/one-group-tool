# ONE Group Tools

**Smarter Decisions. Bigger Profits.**

ONE Group Tools is a comprehensive, static web application for the SimCompanies business simulation game. Built by ONE GROUP OF ENTERPRISES.

## Features

- **16+ Tools:** Profit Calculator, Exchange Tracker, Production Chain, Strategy Advisor, and more.
- **Live Data:** Fetches real-time prices from SimCompanies Public API (v4).
- **Dual Theme:** Native Dark and Light modes with instant toggle.
- **Enriched Analysis:** Historical pricing data integration via SimCoTools.
- **Local Caching:** Fast performance with zero server dependency.
- **Privacy:** No accounts required. All data stored in LocalStorage.

## Usage

1. Open `index.html` in any modern web browser.
2. Select a tool from the dashboard.
3. Configure settings (Realm, Tax Rate, etc.) in the Settings page.

## Deployment

This site is designed to be hosted on **GitHub Pages** or any static file host.

1. Upload all files to your repository.
2. Enable GitHub Pages in Settings -> Pages.
3. Select `root` or `docs` folder as source.

## Tech Stack

- HTML5 / CSS3 (Vanilla, CSS Variables)
- JavaScript (ES6 Modules)
- Chart.js (Visualization)
- Feather Icons (UI Icons)
- html2canvas (Export)
