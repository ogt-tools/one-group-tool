API Documentation

Overview

ONE Group Tools consumes the public SimCompanies API v4 (and legacy v2/v3). We do not provide our own backend API. All data is fetched directly from SimCompanies servers and cached locally in your browser (LocalStorage) to respect rate limits.

SimCompanies API

  • GET /v2/en/encyclopedia/resources/

    Base resource definitions.

  • GET /v2/en/exchange/{id}/

    Live market listings.

  • GET /v2/en/companies/{id}/

    Public company profiles.

Enrichment (SimCoTools)

We use simcotools.app as an enrichment layer for historical data.

  • GET /api/v3/resources/{id}?realm={r}

    Historical price averages (30d) and trends.

Rate Limiting Policy

We strictly adhere to SimCompanies API guidelines. All requests are throttled (min 500ms spacing) and cached. Exchange data is cached for 5 minutes. Static data is cached indefinitely until cleared.
